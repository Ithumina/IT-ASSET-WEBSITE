# TechFix IT Support & Asset Store

This is a premium, multi-page website designed for an IT Support professional. It features a modern design with glassmorphism effects, a dark theme, and a fully functional frontend store.

## Features

- **Responsive Design**: Works perfectly on mobile, tablet, and desktop.
- **Modern UI**: Uses Tailwind CSS with a custom configuration for a premium dark mode look.
- **Multi-Page Layout**:
  - `index.html`: Landing page with hero section and feature highlights.
  - `services.html`: Detailed list of IT support solutions.
  - `shop.html`: A functional product catalog with category filtering and a shopping cart.
  - `contact.html`: Contact form and business information.
- **No Backend Required**: Runs directly in the browser. No Node.js installation needed.

## key Technologies

- **HTML5**: Semantic structure.
- **Tailwind CSS (CDN)**: For styling. No build step required.
- **Vanilla JavaScript**: For interactions (mobile menu, cart logic).
- **FontAwesome**: For icons.
- **Google Fonts**: Uses 'Inter' for a clean, modern typeface.

## How to Use

1. Simply double-click on `index.html` to open the website in your browser.
2. Navigate through the menu to explore Services and the Shop.
3. In the Shop, you can add items to the cart, filter by category, and view your cart breakdown. The cart data is saved in your browser, so it persists even if you refresh the page!

## Customization

To change the content, simply open the `.html` files in any text editor (like Notepad or VS Code) and edit the text.
- To add more products, edit the `products` array in the `<script>` section of `shop.html`.
- To change images, replace the `src` attributes with your own image URLs.
